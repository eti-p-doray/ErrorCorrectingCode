Detailed Description

This class contains the abstract implementation of the belief propagation decoder.

This algorithm is used for decoding in an LdpcCodec. The reason for this class is to offer an common interface of bp decoders while allowing the compiler to inline implementation specific functions by using templates instead of polymorphism.

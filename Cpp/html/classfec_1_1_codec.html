Detailed Description

This class represents a general encoder / decoder.

It offers methods to encode and to decode data given a codec structure. Several specialisations of this class define different codec types.

enum fec::Codec::DecoderAlgorithm

This enum lists the types of decoder algorithm.

Enumerator
Exact 

No approximation is used and the L-values are computed in logarithmic domain.

Linear 

A lookup table with linear interpolation is used to compute the correction term.

Approximate 

The correction term is discarded.

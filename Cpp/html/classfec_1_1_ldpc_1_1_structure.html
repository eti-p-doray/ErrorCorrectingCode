Detailed Description

This class represents a ldpc code structure.

It provides a usefull interface to store and acces the structure information.

Ldpc code structure constructor.

Constructs an ldpc code structure following a given ldpc matrix. A generator matrix is not needed to execute encoding. The matrix does not need to be triangular in shape or be invertible. The ldpc matrix will be transformed (if needed) to allow in-place encoding using its partial triangular form.

Parameters
HLdpc matrix describing the code
iterationsMaximum number of iteration in belief propagation decoding. The decoder can stop before the maximum number of iteration if the msg is consistent.
typeDecoder algorithm used

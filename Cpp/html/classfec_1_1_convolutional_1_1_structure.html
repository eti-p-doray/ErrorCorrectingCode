Constructor.

Constructs an ldpc code structure following a trellis structure.

/*
We are creating a trellis structure with 1 input bit.
The constraint length is 4, which means there are 3 registers associated
with the input bit.
There are two output bits, the first one with generator 4 (in octal) associated
with the input bit.
*/
fec::Trellis trellis({4}, {{013, 017}}, {015});
/*
The trellis is used to create a code structure.
We specify that one bloc will conatins 1024 branches before being terminated.
*/
auto encoder = fec::Convolutional::EncoderOptions(trellis, 1024).termination(fec::Convolutional::Tail);
auto decoder = fec::Convolutional::DecoderOptions().algorithm(fec::Codec::Approximate);
Parameters
trellisTrellis structure to follow
blocSizeNumber of branch in one bloc before being terminated
endTypeTrellis termination type
typeAlgorithm use in app decoding

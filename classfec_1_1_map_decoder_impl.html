Detailed Description

template<typename A>
class fec::MapDecoderImpl< A >

This class contains the concrete implementation of the map decoder.

This algorithm is used for decoding with a-priori information in a ConvolutionalCode. The reason for this class is to offer an common interface of map decoders while allowing the compiler to inline implementation specific functions by using templates instead of polymorphism.

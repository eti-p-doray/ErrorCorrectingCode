Detailed Description

template<typename A>
class fec::BpDecoderImpl< A >

This class contains the concrete implementation of the belief propagation decoder.

This algorithm is used for decoding in an LdpcCode. The reason for this class is to offer an common interface of bp decoders while allowing the compiler to inline implementation specific functions by using templates instead of polymorphism.

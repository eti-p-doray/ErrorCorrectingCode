Detailed Description

This class represents a general encoder / decoder.

It offers methods to encode and to decode data given a code structure.

Member Enumeration Documentation

enum DecoderAlgorithm
Enumerator
Exact 

No approximation is used and the L-values are computed in logarithmic domain.

Linear 

A lookup table is used.

An approximation is used.

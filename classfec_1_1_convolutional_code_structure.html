Detailed Description

This class represents a convolutional code structure.

It provides a usefull interface to store and acces the structure information.

Member Enumeration Documentation

enum TrellisEndType

Trellis termination types.

This specifies the type of termination at the end of each bloc.

Enumerator
PaddingTail 

The state is brought to zero by implicitly adding new msg bit.

Truncation 

The state is forced to zero by truncating the trellis.

enum DecoderType

Decoder type This specifies the decode algorithm used in app decoding.

Enumerator
LogMap 

No approximation is used and the L-values are computed in logarithmic domain.

MaxLogMap 

The maxlog approximation is used.

Detailed Description

This class represents a general encoder / decoder.

It offers methods to encode and to decode data given a codec structure. Several specialisations of this class define different codec types. Methods of this class are implemented in cpp using mex to provide good performances. This class directly handles the allocated ressources in cpp.

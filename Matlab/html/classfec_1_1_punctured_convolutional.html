function fec::Convolutional::puncturing ( in  self,
in  varargin 
)
inherited

Creates a permutation of parity bits from Convolutional::PunctureOptions structure, object or NAME, Value list.

The permutation can then transform (Permutation::permute() and then Permutation::dePermute()) the parity bits generated by the same Convolutional Codec. Note that the this function will not make the objet to apply the puncture pattern. You will have to use the created permutation to Permutation::permute() after the encode() method and Permutation::dePermute() before the decode() method. If you want to create a Convolutional Codec with a single puncturing permutation, see PuncturedConvolutional.

Parameters
punctureOptionsstructure, object or Name, Value list containing the permutation properties.
Returns
Generated permutation that will apply the specify punctureOptions
